# simple-cosmosdb-mongodb

🪐🪐🪐 CosmosDB(MongoDB)をAzure上に構築してみる。  
Let's NoSQL!  

## 実行方法

最初に`./terraform`ディレクトリに移動して、以下のコマンドを実行します。  

```shell
terraform init
terraform plan
terraform apply
```

---

`./src`ディレクトリに移動します。  
`.env.example`をコピーして`.env`を作成し、接続文字列を記述します。  
続けてプログラムを実行します。  

```shell
dotnet run
```
